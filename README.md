# 10,000 Hours (iOS)

10,000 Hours is an iOS app designed to log time spent practicing and
developing skills over long periods of time. The app focuses on clarity and
consistency, providing a simple way to record hours without unnecessary
distraction or gamification.

It was built as a personal tool to better understand how effort accumulates
over months and years, and to support deliberate, sustained practice.

## Features
- Track hours spent across multiple skills or activities
- Simple session-based time logging
- Clear total hour summaries
- Persistent local data storage
- Clean, minimal interface focused on usability

## Motivation
The project began after using an existing time-tracking app for several years.
Early on, the interface on that app didn’t bother me, but as I started developing my own
apps, I became more aware of how much visual and functional clutter I was
tolerating. Over time, that friction became noticeable enough that I found
myself wanting a version of the app that focused almost entirely on logging
time, without additional features or distractions. That impulse eventually
led me to build my own.

## Tech
- Swift
- SwiftUI
- Xcode

## Design Notes
- Designed for long-term use rather than short-term goals
- Minimal UI to reduce visual noise
- Emphasis on minimalism over analytics or visualization

## Status
Personal utility project. The app functions as intended and continues to be
used regularly, so it has not required significant updates. Shared here as a
stable snapshot of an actively used tool.
